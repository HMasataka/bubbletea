# bubbletea

A Go project.

## Getting Started

### Prerequisites

- Go 1.19 or later

### Installation

```bash
git clone bubbletea
cd bubbletea
go mod tidy
```

### Usage

```bash
go run main.go
```

### Build

```bash
go build -o bubbletea
./bubbletea
```
